# 빙봉의 로켓

[캐릭터 선택] <br/>
낮은 목소리 => 빙봉 <br/>
높은 목소리 => 조이


[시동걸기] <br/>
소리를 내면 로켓이 떠오른다.


[비행] <br/>
높은 소리를 낼수록 높이 올라간다. 

별에 닿으면 점수가 +10 된다. <br/>
구름에 부딪히면 점수가 -1 된다. 

달까지 무사히 도달하기를!
